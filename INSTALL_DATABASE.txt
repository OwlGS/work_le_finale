===========================================
ИНСТРУКЦИЯ ПО УСТАНОВКЕ БАЗЫ ДАННЫХ
Портал "Корочки.есть"
===========================================

ВАЖНО: Перед выполнением убедитесь, что у вас запущен XAMPP/WAMP/OpenServer
и MySQL сервер работает.

СПОСОБ 1: Через phpMyAdmin (самый простой способ)
---------------------------------------------------

1. Откройте браузер и перейдите по адресу: http://localhost/phpmyadmin

2. В левой панели нажмите на кнопку "Создать" (или "New" если интерфейс на английском)

3. В поле "Имя базы данных" введите: korochki_est

4. В выпадающем списке "Сравнение" выберите: utf8mb4_unicode_ci

5. Нажмите кнопку "Создать"

6. После создания базы данных, выберите её в левом меню (кликните на korochki_est)

7. Перейдите на вкладку "SQL" в верхнем меню

8. Откройте файл database.sql в любом текстовом редакторе (Блокнот, VS Code)

9. Скопируйте ВЕСЬ текст из файла database.sql

10. Вставьте скопированный текст в большое текстовое поле на вкладке SQL

11. Нажмите кнопку "Вперёд" (или "Go") внизу страницы

12. Если всё правильно, вы увидите сообщение об успешном выполнении запросов

13. Проверьте, что созданы таблицы: users, courses, applications, reviews
   (они должны появиться в левом меню под базой данных korochki_est)


СПОСОБ 2: Через командную строку MySQL
---------------------------------------

1. Откройте командную строку (cmd) или PowerShell

2. Перейдите в папку, где установлен MySQL (обычно C:\xampp\mysql\bin)
   Или если MySQL в PATH, просто выполните команду mysql

3. Выполните команду:
   mysql -u root -p

4. Введите пароль (если установлен, если нет - просто нажмите Enter)

5. Выполните команду:
   source C:\Users\Костя\Desktop\экзамен проект\work_le_finale\korochki-est\database.sql

   ИЛИ скопируйте содержимое database.sql и вставьте в командную строку MySQL

6. Проверьте создание базы:
   SHOW DATABASES;
   USE korochki_est;
   SHOW TABLES;


ПРОВЕРКА УСТАНОВКИ
------------------

После выполнения скрипта проверьте:

1. В phpMyAdmin откройте базу korochki_est
2. Должны быть созданы 4 таблицы:
   - users (должен быть 1 администратор)
   - courses (должно быть 3 курса)
   - applications (пока пустая)
   - reviews (пока пустая)

3. Откройте таблицу users - там должна быть запись с логином "Admin"
4. Откройте таблицу courses - там должно быть 3 курса


ВАЖНЫЕ ЗАМЕЧАНИЯ
----------------

- Если при выполнении возникает ошибка про пароль администратора, 
  это нормально - пароль будет правильно хеширован при первом входе через PHP

- Если таблицы не создались, проверьте, что MySQL сервер запущен

- Убедитесь, что кодировка базы данных utf8mb4 для корректной работы с кириллицей
